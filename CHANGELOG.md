# 更新日志

## v1.0.1 (2026-03-29)

### 新增功能

- 重构目录结构为分层架构 (app/core/, app/api/, app/models/, app/services/, app/middleware/)
- 支持 PostgreSQL / SQLite 双数据库
- 实现数据库表结构自动初始化
- 集成 Alembic 数据库迁移功能
- 新增 API Key 认证中间件，支持前端动态开启/关闭
- 新增系统配置功能，支持前端动态修改日志清理策略
- 新增任务名称字段，便于识别任务用途
- 新增任务分类功能，支持按分类过滤任务
- 新增单个任务查询接口 GET /job/{job_id}
- 实现日志定时自动清理功能
- 新增日志统计功能
- 新增健康检查接口 GET /health
- 支持 Docker / docker-compose 部署
- 支持环境变量配置
- 集成 GitHub API 在线检查更新功能
- 优化 @task 装饰器，框架层面统一捕获任务输出

### 问题修复

- 修复暂停任务状态在更新后被重置的问题
- 修复 _task_categories 初始化类型错误
- 修复 get_config() 空字符串返回默认值问题
- 修复 run_os_command 任务无法捕获命令输出的问题
- 修复 Windows 系统下 SQLite 路径格式问题
- 修复项目重构后 APScheduler 任务引用路径变更导致的启动错误
