# 更新日志

## v1.0.2 (2026-03-30)

### 新增功能

- **AI 远程命令能力**
  - 新增 AI 对话接口（`/ai/chat`），支持自然语言管理计划任务
  - 新增 AI 流式对话接口（`/ai/chat/stream`），支持 SSE 流式输出
  - 支持多轮对话上下文记忆，历史消息存储到数据库
  - 支持自然语言触发器解析（"每天早8点" → cron）
  - 任务 ID 自动生成，无需手动指定
  - 新增草案确认机制，所有变更操作需人工确认
  - 草案格式标准化，新增 `display`、`confirm_text`、`cancel_text` 字段，方便前端解析

---

## v1.0.1 (2026-03-29)

### 新增功能

- 新增 API Key 认证中间件，支持前端动态开启/关闭
- 新增单个任务查询接口 GET /job/{job_id}
- 新增健康检查接口 GET /health
- 优化 @task 装饰器，框架层面统一捕获任务输出

### 问题修复

- 修复 get_config() 空字符串返回默认值问题
- 修复 run_os_command 任务无法捕获命令输出的问题
