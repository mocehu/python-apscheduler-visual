# 更新日志

## v1.0.4 (2026-04-01)

### 新增功能

- **异常告警系统**
  - 新增告警渠道管理，支持 Webhook 和邮件两种通知方式
  - 新增告警规则配置，支持单次失败、连续失败N次、执行超时、任务不存在四种触发条件
  - 新增告警历史记录，支持分页查询和筛选
  - 新增告警渠道测试发送功能
  - 任务调度器集成告警检查，自动触发告警通知

- **任务参数文档解析**
  - 增强函数 docstring 解析能力，支持 Google、NumPy、Sphinx/reST 三种格式
  - 任务详情 API 返回参数描述信息，便于前端展示参数说明

### 问题修复

- 修复任务列表 API 返回参数格式问题，`parameters` 字段从对象改为数量值，优化列表展示性能

## v1.0.3 (2026-03-31)

### 新增功能

- **自定义任务能力**
  - 用户可在前端直接编写 Python 任务函数代码，无需修改后台文件
  - 支持创建、更新、删除、启用/禁用自定义任务
  - 自动解析函数参数，返回参数名称、类型、默认值、是否必填等信息
  - 自动检测任务是否被计划任务使用，防止误操作导致任务异常
  - 自定义任务持久化存储到数据库，重启后自动加载恢复
  - 支持热加载任务模块，无需重启服务器

- **自定义任务安全防护**
  - 代码安全检查（AST 语法分析），禁止危险模块和函数
  - 默认限制大幅放宽，只禁止反序列化和代码执行相关模块（pickle、marshal、ctypes 等）
  - 超时保护，防止无限循环或长时间阻塞
  - 用户可自定义禁止模块列表、禁止函数列表、超时时间

- **AI 代码辅助能力**
  - AI 自动生成任务代码，根据用户需求描述生成符合规范的 Python 函数
  - AI 代码审查，分析代码安全性、语法、逻辑问题并给出建议
  - 支持流式响应，解决前端超时问题

- **任务列表增强**
  - 任务列表统一标识是否为自定义任务，方便区分
  - 支持按任务名称查询单个任务详情


### 新增功能

- **AI 远程命令能力**
  - 新增 AI 对话接口（`/ai/chat`），支持自然语言管理计划任务
  - 新增 AI 流式对话接口（`/ai/chat/stream`），支持 SSE 流式输出
  - 支持多轮对话上下文记忆，历史消息存储到数据库
  - 支持自然语言触发器解析（"每天早8点" → cron）
  - 任务 ID 自动生成，无需手动指定
  - 新增草案确认机制，所有变更操作需人工确认
  - 草案格式标准化，新增 `display`、`confirm_text`、`cancel_text` 字段，方便前端解析
